# Media Project

**!Disclaimer!** this Project is only for Educational Purposes only and does not violate any state laws in the the country I live.

## Description

A self-hosted automation stack that lets you request media shows through a web interface, then automatically finds, downloads, and organizes them into a media library you can access it from any device — no manual searching or file management needed.

## How it Works

| Component | Task |
| --- | --- |
|Jellyfin |Media Server |
|Overseer |Request Media via GUI |
|Prowlarr |Search for Seeds and indexers|
|Flare Solverr |Solve Captcha needed for some indexers |
|Recyclarr |pull quality profiles from Trash Guides |
|Trash Guides |publicly maintained Quality Profiles |
|Sonarr |Download Video Files |
|Bazarr |Pull Subtitles |
|qBittorrent |Download Torrent |
